       <strong>     Manual de comandos do bot</strong>

/help ou /ajuda - <i>Exibe manual de comandos</i>
/ofertar - <i>Insere sua oferta de carona no banco de dados</i>
/buscar - <i>Busca no banco de dados caronas disponiveis</i>
/alterar - <i>Altera algum dado no banco de dados</i>
/carona - <i>Incluir seu nome em uma carona ofertada</i>
/cancela - <i>Cancela carona</i>
/excluir - <i>Exclui oferta de carona</i>


       <b>     Sintaxe de comandos</b>

/ofertar <code>ORIGEM DESTINO HORA NºVAGAS VALOR1 VALOR2</code>
/ofertar <code>DATA ORIGEM DESTINO HORA NºVAGAS VALOR1 VALOR2</code>
<b>*DATA</b> --> <i>Use DATA para agendar uma oferta</i>

/excluir <code>HORA</code>
/excluir <code>DATA HORA</code>
<b>*DATA</b> --><i> Use DATA para excluir uma viagem agendada</i>

/carona <code>MOTORISTA HORA</code>
/carona <code>DATA MOTORISTA HORA</code>
<b>*DATA</b> --> <i>Use DATA para inclusão em oferta agendada</i>

/cancela <code>MOTORISTA HORA</code>
/cancela <code>MOTORISTA DATA HORA</code>
<b>*DATA</b> --> <i>Use DATA para cancelar carona agendada</i>

/buscar <code>ORIGEM DESTINO</code>
/buscar <code>PALAVRA</code>
<b>*PALAVRA</b> --><i> data, origem, destino, hora ou nome_motorista</i>

/alterar <code>ORIGEM DESTINO OPCAO ALTERACAO</code>
/alterar <code>ORIGEM DESTINO DATA OPCAO ALTERACAO</code>
<b>*DATA</b> --> <i>Use DATA para alterar oferta agendada</i>
<b>*OPCAO</b> --> <i>data, hora, vagas, origem, destino, valor1 ou valor2</i>
